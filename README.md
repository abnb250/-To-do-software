# 待办软件   

**基本介绍：**

该软件是一个ai编程（vibe coding）项目。使用中国免费的ai开发。从零开始，根据ai的指导一步一步构建项目，构建代码。
#

**软件特色：**

软件还结合了一些高效能生活的特色：
- 优先级：待办事项根据优先级分为四类：重要紧急、重要不紧急、不重要紧急、不重要不紧急。

- 四象限：用户可以根据任务的重要性，分成四宫格。提高日常生活的效能
- 日历属性：待办事项+日历，待办事项不局限于一天的安排，可以进行一个月、两周，一周安排待办事项
- 目标页面：对于没有特定截止时间的任务，可以设置为目标。
#

**主要功能：**

软件的主要功能能满足大部分的日常需要：

- 记录任务：可以记录有截止日期和截止时间的任务

- 记录目标：可以记录没有截止日期的任务
- 培养习惯：可以将要配养的习惯设置为每天的任务，重要不紧急类别
- 提升效能：培养以事项重要性为指标的安排任务视角

# To-do-software

**Basic introduction:**
This software is an AI programming (vibe coding) project. It is developed using free AI technology in China. Starting from scratch, the project and code are built step by step under the guidance of AI.
#
**Software Features:**
The software also incorporates some features for efficient living: 
- Priorities: To-do items are categorized into four types based on priority: important and urgent, important but not urgent, urgent but not important, and neither important nor urgent.

- Four Quadrants: Users can categorize tasks into four quadrants based on their importance, enhancing the efficiency of daily life.
 - Calendar Attributes: Combines to-do lists and calendars, allowing users to schedule to-do items not just for a single day, but also for a month, two weeks, or a week. 
 - Goal Page: Tasks without specific deadlines can be set as goals.
#
**Main functions:**

The main functions of the software can meet most daily needs:

- Task recording: It can record tasks with deadlines and expiration times

- Record Goals: You can record tasks without a deadline. 

- Cultivate Habits: You can set habits to be cultivated as daily tasks, categorized as important but not urgent. 

- Enhance Efficiency: Cultivate a perspective of arranging tasks based on the importance of the matter.
